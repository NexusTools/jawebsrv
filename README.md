JaWebSrv
==============

Java based Modular Multi-Threaded WebServer

Features
--------
 - HTTP Protocol (In Progress)
 - HTTPS Protocol (Planned)
 - SPDY Protocol (Planned)
 - Proxy Module (In Progress)
 - Stream/File Module (In Progress)
 - Virtual Hosts (Planned)
 - CGI (Planned)
